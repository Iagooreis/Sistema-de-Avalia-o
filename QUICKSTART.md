# ⚡ Quick Start

Início rápido em 2 etapas (SQLite - sem instalações extras!):

## 1️⃣ Setup Backend

```bash
cd server
npm run migrate  # Cria banco de dados SQLite automaticamente
npm run dev      # Inicia em localhost:5000
```

## 2️⃣ Setup Frontend

```bash
cd client
npm start        # Abre em localhost:3000
```

## ✅ Verificar

- Backend: http://localhost:5000 (deve retornar JSON)
- Frontend: http://localhost:3000 (deve carregar Landing Page)
